Table Summary

This section of the Offload Status Report displays statistics and database information for all offloaded tables 1, grouped by schema.

Summary-Level Information

By default, summarised Retained, Reclaimable and Offloaded 2 schema sizes are presented, together with the number of offloaded tables in the schema.

Each schema can be expanded by clicking on the icon to display summarised totals of Retained, Reclaimable and Offloaded size and segments 2 for each offloaded table.

Detail-Level Information

If the Offload Status Report is run at detail level, additional information is provided for each table.

First, the Offload Type for the table will be dispayed along with the summary statistics.

Second, more detailed additional information can be accessed by clicking on the icon. A popup window will be displayed with the following tabs:

  • Summary: retained, reclaimable and offloaded totals by size, segments and rows 2.
  • Details: overview of the main source, hybrid and offloaded tables and views.
  • Parameters: overview of the main offload parameters and partitioning characteristics for the source and offloaded tables.
  • Dependent Objects (if applicable): overview of other dependent objects created when the source application table was offloaded.
  • Incremental Update Objects (if applicable): overview of additional database objects created to support Incremental Update.
  • Advanced Aggregation Pushdown Objects (if applicable): overview of additional database objects created to support Advanced Aggregation Pushdown.
  • Join Pushdown Objects (if applicable): overview of additional database objects created to support Join Pushdown.

Search and Sort

Each expanded schema has Search and Sort capabilities.

To Search, begin typing in the search field and dynamic filtering will be applied as you type.

To Sort, click on the icon in the column header you wish to sort by. Click again to reverse the sort direction. The results are initially sorted by table name in ascending order.


1 In-scope schemas and tables are determined by the --schema and --table report options. If no options are provided, the report will include all tables offloaded from the source RDBMS, else only offloaded tables matching either the requested schema, table or both will be displayed.

2 See Database Summary help for a detailed description of the Retained, Reclaimable and Offloaded statistics and Size, Segments and Rows dimensions.
